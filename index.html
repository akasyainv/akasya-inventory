Before using the system, you need to configure your Firebase project. Please follow the steps below:

  1. Go to Firebase Console
  2. Create a new project or use an existing one
  3. Enable Authentication (Email/Password sign-in method)
  4. Create a Realtime Database
  5. Copy your Firebase config object
  6. Paste it into js/database.js file, replacing the placeholder values
After updating the config, refresh this page to continue.
